MacPerl Build Process

These are the install procedures for MacPerl version 5.6.

Ingredients

  • The dmake MPW tool
  • The Patch MPW tool
  • The latest version of a number of packages:
    • GUSI 2 (GUSI_215 or later)
    • sfio (sfio_04Aug99 or later)
    • (Berekeley) db (db_186 or later)
    • XL (XL_15 or later)
  • AEGizmos
  • MoreFiles 1.5
  • IC source
  • A complete installation of MPW
  • A reasonably recent version of the Metrowerks CodeWarrior compilers for MPW (Pro 5 recommended) or of the MPW C/C++ compilers (I used SC 8.8.4f1, MrC 4.1.0f1c1, MWCPPC 2.3.2 build 75, and MWC68K 2.3.2 build 79)
  • STLport 3.12.3 if MrC/SC
  • (Perl and MacPerl source)

Recipe

  1. Install the patch and dmake MPW tools (including BuildProgram and BuildCommands from the dmake distribution).
  2. Create a folder "MacPerl".
  3. Unpack the perl distribution and put it into that folder.
  4. Put either the db and XL distributions or aliases to them into that folder. (IC, MF, AEGizmos)
  5. Unpack the GUSI and sfio distributions and run their installation scripts GUSI_Install.MPW and sfio_Install.MPW.
  6. STLport / CW_Max_NoDup_Update
  7. Edit Makefiles for compilers
  8. Change directory in MPW to :perl:macos: and buildprogram all
  9. Change directory in MPW to :macperl: and buildprogram all
  • Put the the MacPerl distribution (the "macos" folder) inside the perl distribution (perl-5.6.0).
  • Change your directory to the macos folder.
  • Apply the patch by executing the command "patch < MacPerl-patch-5.6.0"
  • Edit the MacPerlConfig.mk file to specify your preferred compilers.
  • Build the "all" target. It will fail at the "x2p" point.

Good luck, and have fun Matthias Neeracher 09Apr00
