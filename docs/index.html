MacPerl

Welcome to the SourceForge home of MacPerl, the Mac OS port of Larry Wall's programming language Perl. MacPerl was written by Matthias Neeracher, and is currently maintained by Chris Nandor.

MacPerl 5.6 is in development (pre-alpha). The latest release of MacPerl is 5.2.0r4.

For questions about MacPerl, use the MacPerl mailing lists (see the macperl-porters list to follow, or be involved in, the development of MacPerl). For bug reports, use the MacPerl bugs page. When you submit bug reports, please be very clear about what version/revision of MacPerl (or other files/programs/modules) you are using.

Please direct general Perl questions and bug reports not to the MacPerl places, but to the appropriate places for Perl itself.
