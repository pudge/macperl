MacPerl

Welcome to the SourceForge home of MacPerl. Please feel free to go over to the MacPerl project page on SourceForge to submit bug reports and patches, check out from CVS, and see the development progress.

MacPerl is the Mac OS port of Larry Wall's programming language Perl. MacPerl was written by Matthias Neeracher, and is currently maintained by Chris Nandor. For questions about MacPerl, use the mailing lists. For bug reports, use the MacPerl bugs page. When you submit bug reports, please be very clear about what version/revision of MacPerl (or other files/programs/modules) you are using. Direct general Perl questions and bug reports not to the MacPerl places, but to the appropriate places for Perl itself.
