MacPerl 5.6 - Read Me

This is still very a work in progress, but there *has* been progress. Please report your success or failure back to macperl-porters@macperl.org, but don't expect me to act too quickly on the latter.

Ingredients

In addition to this distribution, you need:
  • The dmake MPW tool
  • The Patch MPW tool
  • The latest version of a number of packages:
    • GUSI 2 (GUSI_206 or later)
    • sfio (sfio_04Aug99 or later)
    • (Berekeley) db (db_186 or later)
    • XL (XL_15 or later)
  • A complete installation of MPW. You'll probably want to increase the memory partition for it.
  • A reasonably recent version of the Metrowerks CodeWarrior compilers for MPW (Pro 5 recommended) or of the MPW C/C++ compilers (I used SC 8.8.4f1 and MrC 4.1.0f1c1).
  • And, of course, a Perl distribution (5.6 or later).

Recipe

  • Install the patch and dmake MPW tools (including the BuildCommands and BuildProgram scripts in the dmake distribution).
  • Create a folder "MacPerl".
  • Unpack the perl distribution and put it into that folder.
  • Put either the db and XL distributions or aliases to them into that folder. The top level links or folders must be named "db" and "XL", respectively.
  • Unpack the GUSI and sfio distributions and run their installation scripts GUSI_Install.MPW and sfio_Install.MPW (Currently, you have to "directory" to the folder where these scripts are before executing them).
  • Put the the MacPerl distribution (the "macos" folder) inside the perl distribution (perl-5.6.0).
  • Change your directory to the macos folder.
  • Apply the patch by executing the command "patch Edit the MacPerlConfig.mk file to specify your preferred compilers.
  • Build the "all" target. It will fail at the "x2p" point.
Good luck, and have fun
